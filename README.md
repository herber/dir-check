# Dir check
> Check if a path is a directory

## Installation
```bash
npm install --save dir-check
```

## Usage
```javascript
const check = require('dir-check');

newport.then(port => {
  // do something with port
});
```

> Working example in [test.js](https://github.com/tobihrbr/new-port/blob/master/test.js)
